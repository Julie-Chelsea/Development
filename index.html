Learn languages your way

Name offers you an intuitive and free solution to learning languages. We match you to words and phrases that are most relevant to you, and have several resources available to best fit your learning style.

You no longer have to think about whether you have learned something sufficiently, because we use research to ensure that you master the material, and to decide when to remind you to review previous content.

Don't worry about what to learn next, because we have you covered. And if you choose, you can modify your learning path to better fit your needs.

We currently offer:

  • Korean
  • Japanese

Name is (clearly) coming soon!

Julie and Chelsea began working on it on December 21st, 2014.

Name is going to be powered by several things, which means that we're going to be learning, or advancing in, new programming languages and libraries. These are the some of the things that we will be powering Name with:

  • Polymer
  • Django
  • Python
  • MySQL
  • SQL
  • CSS Animations
  • jQuery

Interested in our progress?

Once we learn a bit more of Django, we'll create a mailing list so that you can get key updates, such as when Name is launched, and when it actually is named. Until then, you can just see how Name morphs by visiting this URL.

You can also check out our GitHub repository to see our changes! Everything isn't uploaded/committed to it, primarily due to security reasons and if the content is just imported from somewhere else, but we still try to add as much as possible.
